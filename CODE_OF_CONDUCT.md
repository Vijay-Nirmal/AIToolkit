# Code of Conduct

This repository is intended to be a respectful, professional place to collaborate.

## Expected behavior

- Be respectful in discussions and reviews.
- Focus on technical issues and concrete improvements.
- Assume good intent and ask for clarification before escalating.
- Accept feedback with the goal of improving the project.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Trolling, intimidation, or disruptive behavior.
- Sharing private or sensitive information without permission.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict access when behavior undermines a safe and productive environment.
