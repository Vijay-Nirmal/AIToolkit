DeckDoc Studio

Convert an uploaded .pptx into one compact DeckDoc document, preview the supported subset in the browser, export the supported subset back to .pptx, and learn the language through interactive feature cards. This is still a starter tool: common text, shape, image, table, chart, and reusable layout workflows are supported directly, while fidelity-heavy features stay visible in the spec explorer and fall back to warnings or extensions when needed.
